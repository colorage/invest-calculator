# Investment Calculator

A simple static investment calculator that projects portfolio growth with monthly contributions, yearly growth, and tax assumptions.

## Live site

**https://colorage.github.io/invest-calculator/**

## One-time GitHub Pages setup

Enable Pages in your repository settings (only needed once):

1. Open [Settings → Pages](https://github.com/colorage/invest-calculator/settings/pages)
2. Under **Build and deployment**, set **Source** to **Deploy from a branch**
3. Set **Branch** to `main` and folder to `/ (root)`
4. Click **Save**

GitHub will publish the site within a minute or two. Future pushes to `main` update the live site automatically.

## Local development

Open `index.html` in a browser, or serve the folder with any static file server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000.

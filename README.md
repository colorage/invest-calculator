# Investment Calculator

A simple static investment calculator that projects portfolio growth with monthly contributions, yearly growth, and tax assumptions.

## Live site

After GitHub Pages is enabled, the site is available at:

**https://colorage.github.io/invest-calculator/**

## Local development

Open `index.html` in a browser, or serve the folder with any static file server:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000.

## GitHub Pages

Deployment runs automatically via GitHub Actions on every push to `main`. No build step is required — the site is plain HTML, CSS, and JavaScript.
